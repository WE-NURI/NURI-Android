# 누리
온 세상에 너의 뜻을 펼치라.


## 누리란?
누리는 지역 경제의 선순환과 숨은 관광지를 발굴할 플렛폼입니다. 
여행객들이 자연스럽게 역사화 문화를 습득하며 지식의 폭을 넓히고, 여행의 질을 높입니다.
여행객들이 여행 중 역사, 문화, 특산물을 자연스럽게 접할 수 있도록 할 것입니다.
